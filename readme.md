# Vue r2pt

## Abstract

jQueryを使わず、Vue.jsのみでページトップへ戻るを実装できるかテスト

## Using

1. `git clone`またはzipダウンロード
2. `npm i -D`
3. `./node_modules/sass-basis/src/css/basis.scss`の`@import 'normalize';`を`@import '../../../normalize-scss/sass/normalize';`に変更
4. `npm start`
    - 生成物を削除する場合は`npm stop`
